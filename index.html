设置

在线版长回复请用「自定义代理」：把项目里的 cloudflare-worker.js 部署到 Cloudflare Workers，填入 Worker 地址。

单次回复写到一半就停，通常是输出 token 达到上限。建议选择「尽可能大」。

「仅最近 N 轮」在对话很长时可避免超出模型上下文窗口，并节省输入 token；与单次回复能写多长无关。
